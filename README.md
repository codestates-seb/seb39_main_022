 **1. 위치정보를 확인하기 조차 어려움. (정보접근성 ↓)**                                                                                     

**`전국 전동휠체어 급속충전기 지도 사이트가 존재하지 않음.`**
포털사이트에 “[거주지역]+ 전동휠체어 급속충전기 “ 검색으로,
*이용자가 지자체 사이트와 정보를 하나하나 알아내야 함*.
⇒ 정보취약계층의 고립을 야기.

- **현재 온라인 서비스 운영실황**
`PC/ Web`
네이버 지도로 전국적인 위치확인은 가능하나, 운영여부와 접근성 등 자세한 내용은 알 수 없음.

`Mobile/ App`
안드로이드, 애플 모두 운영중인 서비스가 없음.
협동조합: 휠누리 앱 (마지막 업데이트 2018년)

 **2. 이용정보 안내가 지역마다 제각각. (정보 불평등)**                                                                                       

검색을 해도 위치를 찾아볼 수 없거나,
지도안내 없이 텍스트 정보만 있는 지역이 많다.

`주관적인 생각`
`사실상 서울과 수도권 외에는 지도정도 보급이 미약한 상태로 보여진다.
정보서비스의 질은, 서울시에서 운영하는 웹사이트가 가장 현실적으로 도움이 되고.
그 외 지역은 지자체 사이트에 [지도API + 기본정보(텍스트)]만 보여져도 훌륭하다고 느낄만큼 정보 확인이 어렵다.`

++`충전소의 운영여부, 관리상태, 접근성을 모두 확인할 수 있는 사이트는 서울시가 유일한것 같다.
하지만 가독성이 떨어지고 적극적인 홍보가 이루어지지않아, 사이트 존재여부도 알기 어렵다.
기사를 5개 쯤 읽었을때에 겨우 알게 되었음..`

## 기능 요구사항 
> front-end
 
 
> back-end
 1. 회원가입 & 로그인
 - 세선을 이용하여 회원에 대한 정보를 server에 저장하도록 함 
 2. 회원정보 수정
 - 로그인 한 후 본인에 관련된 이메일, 닉네임, 전화번호등 본인에 관련된 정보를 수정
 3. 로그아웃
 - 로그아웃시 서버에 저장된 세션ID를 삭제하여 연결 해제
 4. 내 주변 충전소 검색
 - 검색창에 도 이름, 도시이름, 구 이름, 도로명 주소, 지번 주소 등 본인이 익숙한 검색어로 검색을 했을때 충전소 검색
 5. 충전소 후기 작성 & 삭제
 - 충전소를 사용해본 경험에 대해 후기를 남길 수 있고, 삭제 할 수 있다
 6. 즐겨찾기
 - 내가 자주 이용하는 충전소를 즐겨찾기로 등록하여 검색을 하지않고 바로 확인 가능
 7. 충전소 추천 & 비추천
 - 내가 이용해본 충전소 중 좋았던 곳에 대해 추천을 할 수 있고, 비추천 할 수 있다
 8. 공공데이터 api 받아오기
 - WebClient를 이용하여 전국 전동휠체어 충전소 위치를 Mysql 데이터베이스에 저장
